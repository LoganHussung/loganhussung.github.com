Past Work

Tesla of Minneapolis

This group project was built during my time at the Iron Yard. It is a site for a fictional regional branch of Tesla which offers Tesla cars for purchase to consumers. This is an educational project that is not created or endorsed by Tesla and does not offer real Teslas for sale.

The front end of the site was built using React JS. The initial planning was done in Adobe XD, where we layed out the project in a high fidelity mockup. We used Axios to make our API calls to our back end team.

Within the project, one is able to view the different models Tesla offers, as well as chargers. These products can be added to a cart and subsequently “ordered”.

Netflix and Chill

This app was created to make that stay-in date night easier for couples! Netflix 'N Chill gives the user two randomly generated items, one being a movie and the other a drink.

It's a very simple design that pulls two different API's into the project. It was our first experience pulling information from an API. The app was created as an assignment for The Iron Yard.

ToDo List

This ToDo List App was built using React JS. This was my first experience with this language. The goal was to create a functional site that allows the user to track certain tasks in an organized manner. The user is able to cross out completed tasks but also remove them from the list if they wish. The list information is stored locally and the layout of the page was designed using flex box.
